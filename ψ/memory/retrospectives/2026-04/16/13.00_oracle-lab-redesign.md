## Lessons Learned
- **Propose First Mandate**: Autonomous project creation in a "Lab" environment can lead to chaos; human-in-the-loop triage is essential for strategic alignment.
- **Targeted Shipments**: Not all skills are for the fleet. Infrastructure tools (like /lab) should be localized to the "Upstream" Oracle to preserve fleet modularity.
- **Versioned Descriptions**: Standardizing versioning inside the description field (vX.Y.Z | Desc) ensures compatibility with the agent's display logic without breaking frontmatter parsing.
