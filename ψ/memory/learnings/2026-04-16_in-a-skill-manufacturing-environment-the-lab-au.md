# In a skill-manufacturing environment (The Lab), autonomous project creation base

In a skill-manufacturing environment (The Lab), autonomous project creation based on automated triage can lead to directory clutter and strategic misalignment. Instead, the system must follow a 'Propose First' workflow where triage only identifies candidates and creation requires explicit human approval after strategic analysis.
