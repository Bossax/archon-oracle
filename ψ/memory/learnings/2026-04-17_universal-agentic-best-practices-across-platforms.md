# Universal agentic best practices across platforms (Claude Code, Gemini CLI) conv

Universal agentic best practices across platforms (Claude Code, Gemini CLI) converge on the critical importance of:
1.  **Concise, Layered Instruction Sets**: Keep primary instruction files (e.g., CLAUDE.md, GEMINI.md) under 60 lines. Bloated instructions degrade agent performance ("Agent Dumb Zone").
2.  **Explicit Explore-Plan-Implement-Verify (EPIV) Workflows**: Mandate a structured approach where exploration and planning precede implementation, and every change is empirically verified.
3.  **Proactive Context Management**: Manually trigger context compaction at ~50% usage to avoid performance degradation.
