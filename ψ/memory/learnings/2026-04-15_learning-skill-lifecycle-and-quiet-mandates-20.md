# Learning: Skill Lifecycle and Quiet Mandates (2026-04-15)

**Pattern**: Using a three-tier lifecycle (Ship/Install/Execute) and preserving mandate purity.

1. **The Lifecycle**: Archon (Upstream) ships the "Form" to the agent's nervous system. The Recipient (Downstream) installs the skill via their own "Will." The agent executes the skill repeatedly.
2. **Quiet Mandates**: Behavioral rules (GEMINI.md) should only contain long-term laws. Transient manuals (installation commands) belong in Signal Tickets to prevent contextual rot. 
3. **Strategic Precedence**: A Design Requirement Document (DRD) synthesized from signals is required before any code is shipped.

This reinforces the Archon identity: Architect of logic, Registry of form.
