# Learning: Human-in-the-Loop Skill Incubation (2026-04-16)

**Pattern**: Propose First Mandate
**Concepts**: governance, human-in-the-loop, skill-manufacturing, oracle-lab
**Source**: rrr: Archon
**Project**: github.com/Bossax/archon-oracle

### The Pattern
In a skill-manufacturing environment (The Lab), autonomous project creation based on automated triage can lead to directory clutter and strategic misalignment. Instead, the system must follow a "Propose First" workflow.

### Key Insights
1. **Discovery vs. Execution**: Triage should only identify "Candidates" (potential projects) without modifying the registry or file system.
2. **Strategic Analysis**: Before incubating, the system must analyze whether an issue/signal belongs in an existing skill (improvement) or requires a new one (expansion).
3. **Human Approval**: Every new skill project must be explicitly approved by the human via an incubate command after reviewing the strategic proposal.

This ensures that the fleet's evolution remains intentional, modular, and free of "Skill Bloat."
